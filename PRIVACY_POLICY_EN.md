# Privacy Policy

Last updated: January 7, 2025

## Introduction

"Spreadsheet to Markdown" (hereinafter "the Extension") is committed to respecting and protecting user privacy. This Privacy Policy explains how the Extension handles data.

## Data Collection

**The Extension does not collect any personal data.**

## Data Processing

The Extension performs the following operations:

1. **Clipboard Access**: Reads cell data selected in spreadsheets and writes the Markdown-converted result to the clipboard.
2. **Local Processing**: All data conversion processing is performed locally within the user's browser.

## Data Storage

- The Extension does not store any user data.
- No data is transmitted to external servers.
- Clipboard data is discarded when the extension popup is closed.

## Required Permissions

The Extension uses the following permissions:

| Permission | Purpose |
|------------|---------|
| `clipboardRead` | Read data copied from spreadsheets |
| `clipboardWrite` | Write converted Markdown to clipboard |
| `activeTab` | Copy cell selection in the current tab |
| `scripting` | Execute copy operations on spreadsheet pages |

## Third-Party Sharing

The Extension does not share any data with third parties.

## Contact

For questions about this Privacy Policy, please contact us through the GitHub repository Issues.
