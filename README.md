# Spreadsheet to Markdown

スプレッドシートで選択したセルを、Markdownテーブル形式に変換するChrome拡張機能です。

[English](#english) | [日本語](#日本語)

---

## 日本語

### 機能

- スプレッドシートで選択中のセルを自動的にMarkdownテーブルに変換
- 変換結果を自動でクリップボードにコピー
- ヘッダー行なしでのコピーも可能
- 日本語・英語対応

### 対応サービス

- Google スプレッドシート
- Microsoft Excel Online
- Microsoft 365

### 使い方

1. スプレッドシートでセルを選択
2. 拡張機能のアイコンをクリック
3. 自動的にMarkdownテーブルに変換され、クリップボードにコピーされます

### 変換例

**スプレッドシート:**
| 名前 | 年齢 | 職業 |
|------|------|------|
| 田中 | 30 | エンジニア |
| 佐藤 | 25 | デザイナー |

**変換後のMarkdown:**
```markdown
| 名前 | 年齢 | 職業       |
| ---- | ---- | ---------- |
| 田中 | 30   | エンジニア |
| 佐藤 | 25   | デザイナー |
```

## English

### Features

- Automatically convert selected cells in spreadsheets to Markdown tables
- Auto-copy converted results to clipboard
- Option to copy without header row
- Supports Japanese and English

### Supported Services

- Google Sheets
- Microsoft Excel Online
- Microsoft 365

### How to Use

1. Select cells in your spreadsheet
2. Click the extension icon
3. The cells are automatically converted to a Markdown table and copied to your clipboard

### Conversion Example

**Spreadsheet:**
| Name | Age | Job |
|------|-----|-----|
| John | 30 | Engineer |
| Jane | 25 | Designer |

**Converted Markdown:**
```markdown
| Name | Age | Job      |
| ---- | --- | -------- |
| John | 30  | Engineer |
| Jane | 25  | Designer |
```
